### Funcionalidades
- [x] Cadastrar usuário
- [x] Editar informações do usuário
- [x] Listar usuários
- [x] Paginação
- [x] Filtro por idade, região e estado civil

- [React-Icons](https://react-icons.github.io/react-icons/)

### Como usar
````bash
# Clone o repositório
$ git clone https://github.com/Tsugami/Softwrap softwrap

# Entre na pasta
$ cd softwrap

# Instale as dependencias
$ yarn install

# Inicie o servidor
$ yarn server start

# Inicie o site
$ yarn client start
````
